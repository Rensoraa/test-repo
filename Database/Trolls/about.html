About This Project

This website provides a community-driven database of Steam profiles, focusing on player behavior and in-game offerings. The goal is to help players identify trends and patterns within the community.

All data is fetched through the Steam API using a Cloudflare Worker proxy - ensuring both speed and privacy. Users can submit Steam IDs along with a reason; every submission is reviewed before being displayed.

A Word from the Creator

This project was created by Rensora out of frustration with players who disrupt farming sessions and ruin the experience for others.

Feel free to reach out on Steam if you have questions, feedback, or suggestions. Thank you for contributing and helping keep this community resource accurate and fair!
